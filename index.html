Welcome to my SampleWebDemo Project.

This project demonstrates the simple implementation of developing a web based application using integration of Struts2 and Spring integration. The code of the project is hosted on GitHub and can be forked anytime to become the contributor. This project not only shows the working application but also shows the usage of different frameworks used to maintain the quality of the project source code using PMD, FindBugs, CheckStyle etc.
